Product

How it works

Search-engine architecture.

A connector layer keeps the index live, hybrid retrieval finds the right passage, and every answer is generated with mandatory citations.

Connector layer

Webhooks & the Events API stream Slack and GitHub in real time; scheduled crawls keep Notion current.

Hybrid retrieval

BM25 keyword search and pgvector semantic search combine with Reciprocal Rank Fusion for precise results.

Cited generation

RAG with mandatory citations and source-aware chunking, tuned differently per source type.
